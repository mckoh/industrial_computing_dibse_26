# Transfer Learning

In diesem Megatutorial greifen wir noch einmal den Anwendungsfall aus dem 2. Megatutorial auf. Wir haben dort ein eigenes CNN entwickelt, um unseren Bilddatensatz zu verarbeiten. Nun wollen wir unseren Code aber auf eine neue Ebene heben. Wir möchten ein Netz entwickeln, das einerseits unser Training erleichtert (indem wir vorgefertigte Komponenten verwenden) und andererseits unsere Klassifikationsergebnisse verbessert (indem wir die Feature-Extraktion besser aufstellen).

## Aufgabenstellung

* [ ] Erstelle eine Lernumgebung, um ein Deep-Learning-Modell zu entwickeln.
* [ ] Recherchiere und implementiere ein Netz, das bereits vortrainiert ist und für unsere Zwecke eingesetzt werden kann.
* [ ] Passe das Netz soweit erforderlich an und mache es trainingsbereit.
* [ ] Trainiere das Netz.
* [ ] Evaluiere die Ergebnisse deines Trainings.
